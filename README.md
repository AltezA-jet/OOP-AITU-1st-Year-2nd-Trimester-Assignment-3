# OOP-AITU-1st-Year-2nd-Trimester-Assignment-3
Project Overview
This project represents an Object-Oriented Music Library API developed in Java, connected to a PostgreSQL database using JDBC.
It provides basic CRUD functionality (create, read, update, delete) for managing songs and playlists.
The project demonstrates advanced OOP principles, JDBC operations, and custom exception handling within a structured multi-layered architecture.
